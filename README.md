# k-shiraishi Claude Code Plugins

Personal Claude Code plugins marketplace.

## Installation

```bash
# Add marketplace
/plugin marketplace add k-shiraishi/claude-code-marketplace

# Install plugins
/plugin install ai-delegator@k-shiraishi-plugins
```

## Plugins

### ai-delegator

Gemini/Codexに相談して多角的な視点を得るプラグイン。

**使い方:**
- `@ai-advisor` で明示的に呼び出す
- 設計検証、コードレビュー、技術調査に活用

**前提条件:**
- `gemini` CLI がインストールされていること
- `codex` CLI がインストールされていること
